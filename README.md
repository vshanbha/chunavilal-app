## Getting Started as developer

### Prerequisites

Python version 3.12

### Installation
Create Python venv
Open a terminal and run:

```bash
$ python3.12 -m venv .venv
$ source .venv/bin/activate
$ pip install -r requirements.txt
$ streamlit run streamlit_app.py
```
